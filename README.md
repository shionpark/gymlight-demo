# Gymlight Manager (Demo)

헬스장 통합 관리 시스템 - 프론트엔드 데모 버전

## 🎯 소개

MSW(Mock Service Worker)를 활용한 프론트엔드 데모 애플리케이션입니다.  
 백엔드 없이 모든 UI/UX를 체험할 수 있습니다.

## ⚡ 빠른 시작

### 설치

```bash
yarn install
```

### 실행 (MSW 모드)

```bash
yarn dev:mock
```

## 🔐 데모 계정

|   역할   | 접근 가능 메뉴                                     |
| :------: | -------------------------------------------------- |
|  관리자  | 공통 메뉴 + 직원/팀/통계/회계 관리 + **지점 관리** |
|  매니저  | 공통 메뉴 + 직원/팀/통계/회계 관리                 |
| 트레이너 | 공통 메뉴 + 나의 업무                              |
|    FC    | 공통 메뉴                                          |

> **공통 메뉴**: 대시보드, 공지사항, 회원/상품/락커/출석 관리

로그인 화면에서 역할별 버튼 클릭 시 해당 역할로 로그인됩니다.

## 🛠 기술 스택

- React 18 + TypeScript
- Recoil (전역 상태)
- React Query (서버 상태)
- MSW (API Mocking)
- Emotion (스타일링)

## 📁 프로젝트 구조

```bash
src/
├── features/ # 도메인별 기능 (회원, 상품, 출석 등)
├── components/ # 공통 컴포넌트
├── pages/ # 페이지 컴포넌트
├── mocks/ # MSW 핸들러 및 Mock 데이터
│ ├── handlers/ # API 핸들러
│ └── data/ # Mock 데이터
└── ...
```

## ⚠️ 참고사항

- 데모 버전으로 데이터 생성/수정/삭제는 실제로 저장되지 않습니다
- 새로고침 시 초기 데이터로 리셋됩니다
- 실제 서비스 버전은 링크 참고
